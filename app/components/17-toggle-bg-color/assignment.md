# 17 - Toggle Background Color and Search

In this assignment, you will build a component that allows you to toggle the background color and includes a search input field.

## Requirements:
- Create a `ToggleBgColor` component.
- The component should have a button that toggles the background color of a container div between two colors (e.g., light and dark mode).
- The component should include a search input field.
- Use the `useState` hook to manage the background color state and the value of the search input.
- The component should be styled to be visually clear and user-friendly.
