# Regular Expression Lecture

This section covers the basics of regular expressions in JavaScript, including:
- Syntax and Modifiers
- Search Methods (`match`)
- Ranges (`[a-z]`, `[0-9]`, `[^...]`)
- Character Classes (`.`, `\w`, `\W`, `\d`, `\D`, `\s`, `\S`, `\b`, `\B`)
- Quantifiers (`+`, `*`, `?`, `{n}`, `{n,}`, `{n,m}`)
- Anchors (`^`, `$`) and Lookarounds (`?=`, `?!`)
- `replace` and `replaceAll` methods

The `RegexTester.tsx` component demonstrates these concepts with various examples.
