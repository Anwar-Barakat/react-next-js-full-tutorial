# Assignment 27: User Registration Form with React Hook Form

This assignment focuses on building a user registration form using `react-hook-form` for efficient form management and validation.

## Task

Implement a `UserRegistrationForm` component that includes:

1.  **Form Fields:**
    *   Full Name (text input)
    *   Email (email input)
    *   Password (password input)
    *   Confirm Password (password input)
2.  **Validation:**
    *   All fields are required.
    *   Email must be a valid email format.
    *   Password must be at least 8 characters long, contain at least one uppercase letter, one lowercase letter, one number, and one special character.
    *   Confirm Password must match the Password field.
3.  **Form Submission:**
    *   Display a success message upon successful submission, showing the submitted data (excluding passwords).
    *   Display validation error messages for each field.

## Learning Objectives

*   Understand and implement `react-hook-form` for form management.
*   Apply various validation rules using `react-hook-form`.
*   Handle form submission and display feedback to the user.
*   Integrate form components with a Next.js application.
