# Assignment: Testing with Jest and JSDOM

This assignment focuses on setting up and writing unit tests for React components using Jest as the test runner and JSDOM for a browser-like environment.

## Task

1.  Create a simple React component (e.g., a button, a display component).
2.  Write a test file for this component using Jest and JSDOM.
3.  Ensure the tests cover basic rendering and interaction (if applicable).
